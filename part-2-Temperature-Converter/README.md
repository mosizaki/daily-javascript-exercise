A simple temperature converter built with vanilla JavaScript.
It allows users to convert temperatures between Celsius and Fahrenheit with basic input validation.

## Features
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Input validation for non-numeric values
- Error message for invalid input
- Reset button to clear inputs

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## How to Run
1. Clone the repository
2. Open `index.html` in your browser

## What I Learned
- Handling user input and validation in JavaScript
- Detecting invalid numeric values (`NaN`)
- Structuring small functions for readability
- Managing UI state based on user input

## Edge Cases Handled
- Empty input clears the other field
- Invalid input does not trigger conversion
